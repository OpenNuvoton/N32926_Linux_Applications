Run make to build all.
The test input file test.pcm is located in the subfolder pattern, user could copy it into the same folder of AACRecorder and run AAC Recorder, "./AACRecorder output_AACfile input_PCMfile". 
The current input file format is PCM, 16KHz, mono channel.
UART log shows "AACEnc_Initialize: Using H/W IP", it means to use AAC IP. If not, it uses software solution.
